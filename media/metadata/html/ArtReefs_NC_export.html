ArtReefs_NC
Geospatial_Data_Presentation_Form: vector digital data
Description:
Abstract: 
This shows the near shore and offshore artificial reefs off the coast of North Carolina.
Purpose: 
This GIS dataset was created for purpose of describing artificial reef zones for the state of North Carolina so that these data can be considered with respect to other data on coastal resources and activities.

Process_Description:
