Data Registry

The data registry provides direct access to data currently available through MarineCadastre.gov. Users of the registry can filter the data by provider, thematic category, and service type. If you are looking for a data set that is currently not available on MarineCadastre.gov, please email us.

Sign up to receive email updates on changes to our map services.

So What?

The “So What?” series explains the importance of various data sets within MarineCadastre.gov and why they should be considered when determining ocean use.
